# Banker's Algorithm Web App

This project simulates the Banker's Algorithm used for deadlock avoidance in Operating Systems.

## Features
- Input number of processes and resources
- Enter Allocation, Max, and Available matrices
- Displays Safe Sequence or Unsafe State
